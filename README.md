# Simple Python Web Server

This project is a minimal web server written in Python. It is based on the tutorial from *The Architecture of Open Source Applications (500 Lines or Less)* and was recreated for learning purposes.

The goal of this project is to understand how web servers work at a low level, without using frameworks.

---

## Features

- Handles basic HTTP requests (GET)
- Serves static HTML files
- Uses Python sockets for communication
- Lightweight and easy to understand

---

## Inspiration

This project is based on the following tutorial:  
https://aosabook.org/en/500L/a-simple-web-server.html

The code has been recreated to better understand the inner workings of web servers.
